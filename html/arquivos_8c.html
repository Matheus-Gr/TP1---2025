◆ gerarArquivoBinario()

void gerarArquivoBinario ( const char * caminhoArquivo,
int quantidadeRegistros,
int situacao,
const char * arquivoIndices )

Gera um arquivo binário de registros, ordenado, reverso ou aleatório.

O arquivo gerado terá quantidadeRegistros registros do tipo Registro. Se situacao == 1, gera registros ordenados crescentes. Se situacao == 2, gera registros ordenados decrescentes. Se situacao == 3, usa um arquivo de índices para ordem aleatória.

Parâmetros
caminhoArquivoCaminho do arquivo binário a ser gerado.
quantidadeRegistrosQuantidade de registros a serem gerados.
situacao1 (ordenado), 2 (reverso), 3 (aleatório).
arquivoIndicesCaminho do arquivo de índices aleatórios (usado se situacao == 3).
